
# RNS RNode interface
For a LoRa based Reticulum data exchange.

## How to build
```bash
$ docker build . -t rnsd-docker:latest
```

## How to launch
* Use `docker-compose`
```bash
$ docker-compose up
```

## How to connect
Connect a RNode using USB, verify the device path (e.g `/dev/ttyACM0`,...)
